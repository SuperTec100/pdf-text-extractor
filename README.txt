PASSOS PARA HOSPEDAR NO RENDER

1. Crie um repositório no GitHub e envie os arquivos deste projeto.
2. Acesse o site https://render.com e crie uma conta (ou entre).
3. Crie dois serviços:
   a. Um Web Service para o backend (Node.js), usando a pasta /server.
   b. Um Static Site para o frontend (React), usando a pasta /client.
4. Altere no App.jsx a URL do fetch para apontar para a URL do backend no Render.
5. Pronto! A aplicação estará funcionando.
